# Kappa\Forms

Collection of custom input controls

## Requirements

* PHP 5.3.3 or higher
* [Nette\Forms](https://github.com/nette/forms) 2.2.* or higher

## Installation:

The best way to install Kappa\Forms is using [Composer](https://getcomposer.org/)

```bash
$ composer require kappa/forms:@dev
```
